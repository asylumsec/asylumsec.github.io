What is the purpose of this site?

The documentation contained here is learned from field for enviroments like the Department of Defense.

Here you will find code snippets, procedures for various software, and random bits of things picked up along the way.

  • This site is maintained by the Red Hat DoD Solution Architect Team
  • For offical documentation on Red Hat products, please see Red Hat Documentation
  • If you want an offical answer to your question, please open a support ticket. Red Hat Support

 

This site is produced using AsciiBinder, which is an awesome project.

AsciiBinder is for documenting versioned, interrelated projects.

  • Source your docs in AsciiDoc
  • Manage doc changes and doc versions with git
  • Have the ability to conditionalize topics and produce different distributions of the docs based on those conditions
